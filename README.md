# altinn-admin

[![Build Status](https://travis-ci.com/navikt/altinn-admin.svg?branch=master)](https://travis-ci.com/navikt/altinn-admin)

Application for adminstration of altinn in the role as a serviceowner.

## Getting started


#### Requirements

* JDK 11

#### Build and run tests

`./gradlew build`

#### Running locally

`./gradlew run`
